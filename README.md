# API-Health-Check-Incident-Monitor
Creating a small API with different endpoints. Each endpoint returns different methods, payloads, and response codes.

Loveable was used to create the websie and the APIs interactions with each other.


Seeing the action of the website

- cd project-codebase
- npm install
- npm run dev (Click localhost)
- npx tsx watch server/index.ts (Seperate Terminal)
